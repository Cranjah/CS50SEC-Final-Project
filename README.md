## CS50SEC APPLE SECURITY

### Description:
I proudly present to you my final project for CS50SEC - Harvard's Introduction to Cybersecurity. Inspired by an article of Security Week, I created a presentation about safety features of macOS in general and a full development cycle for a more or less secure system. This presentation reflects in general the necessities of authentication and authorization, encryption, possible vulnerabilities, compliance rules and guidelines, network security, possible emergency responses, as well as third-party management and recovery.
